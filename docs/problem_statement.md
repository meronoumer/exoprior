ExoPrior is a research-software prototype for ranking confirmed transiting exoplanets for potential follow-up characterization.

The problem is not to determine which planets are objectively “best” or “habitable.” The problem is to create a transparent, reproducible ranking pipeline that combines public archive data, feature engineering, uncertainty handling, and sensitivity analysis.

The core research question is:
How can public exoplanet archive data be transformed into a transparent prioritization system for follow-up observation candidates, and how stable are the resulting rankings under different scientific priorities?
